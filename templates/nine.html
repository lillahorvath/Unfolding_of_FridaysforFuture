ABOUT THIS PROJECT

The Fridays for Future environmental movement has achieved unparalleled success in channeling public support against climate change. In this project, I analyzed the representation of the movement on Twitter to gain actionable insights into how to navigate an impactful environmental campaign. For more information about the movement visit the official homepage and follow Greta Thunberg on Twitter.

About the data and code: The data presented in this project reflect the state of the movement up to July 2019. I generated the map based on the number of registered protests available from the Fridays for Future homepage. To acquire the tweets with the hashtag #FridaysForFuture I used Twitter's API. Note that for the language analysis I collected the 5K most and least popular tweets each, published between Spring-Summer 2019. Popularity was defined based on the number of retweets and likes. For data and code visit the project's github repository.
